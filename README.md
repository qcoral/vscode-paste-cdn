# Image paste CDN

When pasting images in markdown files, get it automatically uploaded to the [Hack Club CDN](https://cdn.hackclub.com/)! No longer will you have to deal with an `image.png` file every time you paste a screenshot.

<img src="https://cdn.hackclub.com/019d5bf7-9950-72e9-a18f-22db6d4bf5ad/final.gif" alt="demo" />

Up to 50GB free for teenagers, and 50MB free for anyone else. Get an API key [here](https://cdn.hackclub.com/api_keys), then set `pasteCdn.apiKey` with your API key.
